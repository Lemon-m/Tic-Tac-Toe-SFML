# Tic Tac Toe SFML

Simple C++ Tic Tac Toe game made with SFML library.

## Requirements
- C++17
- SFML 2.5.1 (see setup instructions below)  
- Visual Studio (recommended)

## SFML Setup
Link to the official SFML 2.5.1 setup guide  
[https://www.sfml-dev.org/tutorials/2.5/start-vc.php](https://www.sfml-dev.org/tutorials/2.5/start-vc.php)

## Build Instructions
1. Clone this repository:  
   ```
   git clone https://github.com/Lemon-m/Tic-Tac-Toe-SFML.git
   cd Tic-Tac-Toe-SFML
   ```
2. Open the project in your IDE or create a new one, then add all `.cpp` and `.h` files.  
3. Configure SFML (see guide above).  
4. Link SFML libraries
5. Build and run the project.  

## Run
After building, simply execute the generated binary.
